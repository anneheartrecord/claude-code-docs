# 01 架构总览

## 一句话概括

Claude Code 是一个基于 AsyncGenerator 的 ReAct Agent，通过终端 UI 与用户交互，核心循环是"构建上下文 → 调 API → 执行工具 → 判断是否继续"。

## 技术栈

| 层 | 技术 |
|---|------|
| 运行时 | Bun 1.3.11+ |
| 语言 | TypeScript 6.0.2 |
| UI | React 19.2.4 + ink（终端渲染） |
| CLI | Commander.js 14.0.0 |
| API | Anthropic SDK + AWS Bedrock + Azure + Vertex |
| Lint | Biome 2.4.10 |
| 打包 | Bun 原生 bundler |

## 数据流

一次完整的用户请求经历以下路径：

```
用户输入
  ↓
REPL 捕获 → 判断是 slash command 还是自然语言
  ↓
QueryEngine.query() → 组装 system prompt + 消息历史
  ↓
query loop 开始
  ↓
预取：记忆文件、Skill 列表、Git 状态（缓存命中则跳过）
  ↓
构建 API 请求：system prompt + messages + tools + thinking config
  ↓
流式调用 Anthropic API → 逐 token 渲染到终端
  ↓
解析响应：纯文本 → 直接输出 | tool_use → 进入工具执行
  ↓
工具执行：Pre-Hook → 权限检查 → 并行执行 → Post-Hook
  ↓
tool_result 追加到消息历史
  ↓
压缩判断：token 用量超阈值？→ 微压缩 / Session Memory / Full Compact
  ↓
继续判断：stop_reason === 'tool_use' → 继续循环 | 'end_turn' → 结束
  ↓
输出最终结果
```
